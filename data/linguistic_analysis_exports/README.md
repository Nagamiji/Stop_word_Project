# Linguistic Analysis Exports

This directory contains exported data from the comprehensive linguistic analysis.

## Files:

1. **category_distribution.csv** - Distribution of stopwords by category
2. **category_impact.csv** - Impact of each category on text (frequency analysis)
3. **tiered_recommendations.csv** - Tiered recommendations for stopword removal
4. **ir_correlation.csv** - Correlation between categories and IR performance

## Usage:

These files can be used for:
- Further statistical analysis
- Creating custom stopword lists
- Research on Khmer linguistics
- Optimizing text processing pipelines
